# ElasticSearch.Sample

Integrate .NET Web API application with Elasticsearch

Using "Elastic.Clients.Elasticsearch" Package, Version is "9.2.1"

oR

Using "Elasticsearch.Net" Package, Version is "7.17.5"
With "NEST" Package, Version is "7.17.5"

Also integrate Elasticsearch with Serilog for Centralized Logging

Using "Serilog.AspNetCore" package, Version="9.0.0"
      "Serilog.Sinks.Elasticsearch" package, Version="10.0.0"

## Run Application using Docker Compose

1. Open terminal and run this command
   "docker-compose -f .\docker-compose.yml -f .\docker-compose.override.yml  up -d --build"
